Namespace: [Microsoft.Quantum.Math](xref:Microsoft.Quantum.Math)

```qsharp
function PowI (a : Int, power : Int) : Int
```


## Input

### a : [Int](xref:microsoft.quantum.lang-ref.int)

The number $a$ that is to be raised.
